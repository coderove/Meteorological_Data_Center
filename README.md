## 数据中心项目



### 项目环境

Centos7、gcc/g++ 10.0.2、cmake3.22、mysql5.7.38-1、oracle11gR2

本项目利用CLion ssh远程开发

### 目录结构

~~~C++
bin      ----- 存放项目生成的所有二进制文件
build    ----- 构建程序（暂时没有用到，都是在工程文件目录直接makefile）
idc      ----- 启动脚本文件start.shell,结束所有进行killall.shell以及配置文件和数据处理等。
log      ----- 项目运行产生的日志文件
public   ----- 相关用到的相关操作函数如数据库操作、socket操作、ftp操作。
tempdata ----- 临时数据存放目录
tools    ----- 项目主要程序存放目录
~~~
